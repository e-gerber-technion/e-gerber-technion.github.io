2D Robot Arm Simulator & Kinematics Engine

An interactive web-based simulator demonstrating forward kinematics (FK) and basic motion programming of a planar robot arm. Features workspace visualization and real-time interactive controls. Built purely with HTML5 Canvas and JavaScript, using a Claude Sonnet 4.6 agent running in Antigravity.
